

# Virtual Music Kit — Piano

## Project Overview

This project is a virtual piano — an interactive musical instrument implemented in the browser. Users can play notes using either their mouse or keyboard. The application uses the Web Audio API to manage audio playback and provide low-latency sound response for a smoother playing experience.

- [**Technical Specifications (Rolling Scopes School)**](https://github.com/rolling-scopes-school/tasks/tree/master/stage1/tasks/virtual-music-kit)
- [**Live Demo**](https://khabib1802.github.io/virtual-music-kit/)

## Technology Stack

| Category | Technologies |
|----------|--------------|
| **Markup** | HTML5 (semantic markup) |
| **Styles** | CSS3 / SCSS (variables, mixins, nesting) |
| **Logic** | Vanilla JavaScript (ES6+) |
| **Responsive** | Mobile, tablet, and desktop support (320px to 1440px+) |

## Key Features

- ✅ **Piano keyboard** — play notes by clicking on keys or using computer keyboard
- ✅ **Interactive key press feedback** — visual and audio response when keys are played
- ✅ **Pure JavaScript implementation** — no external libraries or frameworks
- ✅ **Responsive design** — correct display on all device types
- ✅ **SCSS architecture** — organized styles with variables, mixins, and nesting
